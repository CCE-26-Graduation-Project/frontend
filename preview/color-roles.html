Semantic color roles
Every color in the system maps to one of these slots. Hue changes per palette; role never does.
--accent
Subtle accent — image gradients, search focus ring, camera idle pulse. Brand foreground uses navy + cream instead.
--savings
Lowest price found. Used only on prices — never as a CTA fill.
--secondary-accent
Light-blue chip for tags and soft secondary surfaces.
--error
Soft, never aggressive. Warnings + destructive states.
--character
Body fill of Snoop the mascot. Adapts to the palette automatically.
